The ResumeTex API

Anish Sahoo
December 15, 2024

What is this?

This is a fast way to create resumes. All it needs is a GET request with your resume in JSON format and it will generate a nicely formatted Resume for you. You can also generate the Latex file if you want to make any changes manually.

Endpoints

The API has three endpoints:

  • /api/v1/tex - Generates a LaTeX file
  • /api/v1/pdf - Generates a PDF file
  • /api/v1/text - Generates latex in text format (suitable if downloading a file is not intended)

All endpoints accept GET requests with JSON data at https://resumetex.asahoo.dev

How to use?

You can use the API by sending a POST request to https://resumetex.asahoo.dev/api/ with the following JSON body: 

        
{
  "name": "Anish Sahoo",
  "email": "",
  "phone": "123-456-7890",
  "address": "1234 Elm St, Springfield, IL",
  "education": [
    {
      "school": "University of Illinois at Urbana-Champaign",
      "degree": "Bachelor of Science in Computer Science",
      "graduation": "May 2022",
      "gpa": "3.8/4.0",
      "courses": [
        "Data Structures",
        "Algorithms",
        "Computer Architecture"
      ]
    }
  ],
  "experience": [
    {
      "company": "Google",
      "title": "Software Engineering Intern",
      "start": "May 2021",
      "end": "August 2021",
      "description": "Worked on the Google Search team."
    }
  ],
  "projects": [
    {
      "name": "ResumeTex",
      "description": "A fast way to create resumes for CS Majors."
    }
  ],
  "skills": [
    "Python",
    "Java",
    "C++"
  ]
}

The API will return a JSON response with the URL to the generated PDF and tex file.
